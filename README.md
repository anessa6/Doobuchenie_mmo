# Классификация изображений

## Описание
Результат выполнения лабораторной работы по дообучению предобученных моделей из библиотеки timm для задачи классификации изображений.

## Структура проекта
- `data/` — данные и их описание
- `models/` — каталог хранения моделей
- `experiments/` — ноутбуки и скрипт для обучения
- `app/` — локальное приложение для инференса модели (Gradio/Streamlit)
- `requirements.txt` — зависимости проекта
