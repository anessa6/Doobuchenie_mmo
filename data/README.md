# Описание набора данных

## Задача
Классификация изображений на три типа объектов: коты (cat), фрукты (fruit), обезьяны (monkey). 

## Метод сбора
Изображения собраны вручную с помощью сервиса Яндекс.Картинки. Каждое  изображение является общедоступным со свободной лицензией.

## Статистика
- Общее количество изображений: 126 (cat: 42, fruit: 42, monkey: 42).
- Разделение: train - 90 изображения, validation - 21, test - 15.

## Предобработка
- Ресайз: Изображения приводятся к размеру 224x224 (для совместимости с моделями ResNet/EfficientNet).
- Нормализация: Средние значения [0.485, 0.456, 0.406], стандартное отклонение [0.229, 0.224, 0.225] (ImageNet).
- Аугментации (для train): Random Horizontal Flip (p=0.5), Random Rotation (±15°), Color Jitter (яркость/контрастность), Random Crop.
- Для val/test: только ресайз и нормализация.

## Структура
- `data/raw/`: Необработанные изображения, разделенные по классам.
- `data/split/`: Разделенные данные (train/val/test).
